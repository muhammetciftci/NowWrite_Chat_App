# NowWrite
Chat App with MVVM

An example project for my development

<h1>Features</h1>
<h3>
- MVVM Architecture <br>
- Firebase (Realtime-Database, Authentication, Storage and Cloud Messaging) <br>
- Dagger-Hilt <br>
- Retrofit (For fcm notification) <br>
- Coroutines <br> 
- LiveData <br>
- DataBinding <br>
- ViewBinding <br>
- Navigation <br>
- Glide <br>
</h3>
